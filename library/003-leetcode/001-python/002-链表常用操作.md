# 链表常用操作

## 创建链表
可以创建空的双端链表，也可以从一个可迭代对象生成一个链表、链表中的元素即为顺序迭代的元素
```py
from collections import deque

my_linkedlist = deque()
# deque([])

my_linkedlist2 = deque("hello")
# deque(['h', 'e', 'l', 'l', 'o'])

my_linkedlist3 = deque([1, 2, 3])
# deque([1, 2, 3])
```

## 添加元素
```py
my_linkedlist = deque("hello")

my_linkedlist.append('a')
# 在链表右端添加元素'a'，o(1)
# "helloa"

my_linkedlist.appendleft('b')
# 在链表左端添加元素'b'，o(1)
# "bhelloa"

my_linkedlist.insert(1, 'c')
# 在指定索引位置加入一个元素，o(n)（需要顺序找到索引1的位置）
# "bchelloa"
```

## 删除元素
```py
my_linkedlist = deque("hello")

my_linkedlist.pop()
# 删除链表右端的一个元素，o(1)
# "hell"

my_linkedlist.popleft()
# 删除链表左端的一个元素，o(1)
# "ell"

my_linkedlist.remove('l')
# 删除链表从左到右第一个等于指定值的元素，o(n)
# "el"
```

## 访问元素
```py
my_linkedlist = deque("hello")

c = my_linkedlist[2]
# 访问链表指定索引位置的元素，o(1)
```

## 查找元素
```py
my_linkedlist = deque("hello")

index = my_linkedlist.index('l')
# 找到从左到右第一个等于指定值的元素，返回索引，o(n)
# index = 2
```

## 更新元素
```py
my_linkedlist = deque("hello")

my_linkedlist[2] = 'a'
# 更新链表中指定索引位置的元素的值，o(n)
# "healo"
```

## 长度
```py
my_linkedlist = deque("hello")

len(my_linkedlist)
# 链表长度，o(1)
```
